# Installation

Javet Sanitizer supports 4 versions of antlr.

- 4.13.1
- 4.12.0
- 4.11.1
- 4.10.1

## Maven

```xml
<dependency>
    <groupId>com.caoccao.javet.sanitizer</groupId>
    <artifactId>javet-sanitizer</artifactId>
    <version>0.3.0-antlr-4.13.1</version>
</dependency>

<dependency>
    <groupId>com.caoccao.javet.sanitizer</groupId>
    <artifactId>javet-sanitizer</artifactId>
    <version>0.3.0-antlr-4.12.0</version>
</dependency>

<dependency>
    <groupId>com.caoccao.javet.sanitizer</groupId>
    <artifactId>javet-sanitizer</artifactId>
    <version>0.3.0-antlr-4.11.1</version>
</dependency>

<dependency>
    <groupId>com.caoccao.javet.sanitizer</groupId>
    <artifactId>javet-sanitizer</artifactId>
    <version>0.3.0-antlr-4.10.1</version>
</dependency>
```

## Gradle Kotlin DSL

```kotlin
implementation("com.caoccao.javet.sanitizer:javet-sanitizer:0.3.0-antlr-4.13.1")
implementation("com.caoccao.javet.sanitizer:javet-sanitizer:0.3.0-antlr-4.12.0")
implementation("com.caoccao.javet.sanitizer:javet-sanitizer:0.3.0-antlr-4.11.1")
implementation("com.caoccao.javet.sanitizer:javet-sanitizer:0.3.0-antlr-4.10.1")
```
